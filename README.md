# SeaTech Moderne - Site Web

Site web moderne pour l'école d'ingénieurs SeaTech de l'Université de Toulon.

## Structure du projet

- `/src/app/` - Code principal de l'application Next.js
- `/src/app/pages/` - Pages organisées par catégories
- `/public/images/` - Assets images organisés par type
- `/src/components/` - Composants réutilisables (réservé pour développement futur)

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Le site sera accessible sur `http://localhost:3000`

Voir `STRUCTURE.md` pour une documentation complète de l'organisation du projet.
